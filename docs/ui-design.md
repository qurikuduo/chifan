
**交互说明：**
- 从菜品库添加：弹出菜品选择弹窗（支持搜索、标签筛选）
- 新建菜品并添加：跳转到新建菜品页，完成后自动添加到当前菜单
- 每添加一个菜品自动保存（显示"已保存" / "保存中..." / "保存失败"）
- 点击菜品上的 ✕ 可移除
- 可拖拽排序菜品
- 保存草稿：保存但不发布
- 发布菜单：发布后通知所有被邀请的家人

- 限制：单张最大 5MB，JPG/PNG/WebP

## UI 设计要点

1. **移动端优先**：所有页面先设计 375px 宽度，再适配桌面端
2. **颜色方案**：温暖的食物主题色（推荐橙色/棕色系）
3. **卡片式布局**：菜单和菜品都使用圆角卡片
4. **照片突出**：菜品照片是核心视觉元素，尽量大尺寸展示
5. **状态可视化**：菜单状态用颜色和图标区分
   - 草稿: 灰色 📝
   - 选菜中: 蓝色 ⏰
   - 选菜结束: 黄色 📋
   - 烹饪中: 橙色 🔥
   - 已完成: 绿色 ✅
6. **操作反馈**：所有操作都有即时反馈（loading、toast提示等）
7. **空状态**：各列表页面在无数据时展示友好的空状态图和引导
